# 제출용 요약: 자율주행 평가 AI Agent

## 시스템명

AV Evaluation Agent: 시나리오 기반 자율주행차 평가 자동화 Agent

## 한 줄 정의

자연어로 입력된 자율주행 시험 조건을 표준 시나리오 정의서로 변환하고, CARLA/OpenCDA 시뮬레이션 준비와 KPI 산출, 결과 dashboard 생성을 자동화하는 평가기관용 AI Agent 시스템입니다.

## 차별점

1. 사용자의 자연어 명령을 바로 실행하지 않고, 먼저 구조화된 `scenario_definition.json`으로 변환합니다.
2. AI는 판단, 검증, 계획, 설명을 담당하고 KPI 계산은 재현 가능한 Python 코드가 담당합니다.
3. 원본 OpenCDA 파일을 직접 수정하지 않고, run별 복사본과 manifest를 남겨 추적성을 확보합니다.
4. 인지, 제어, 교통 영향성, 주행 안전성을 하나의 평가 체계로 통합합니다.
5. n8n과 LangGraph를 결합해 승인, 반복 실행, 상태 조회, 결과 보고까지 확장할 수 있습니다.

## 연구적 의미

기존 자율주행 평가 과정은 시나리오 작성, 시뮬레이션 실행, 로그 정리, KPI 계산, 보고서 작성이 분리되어 반복 비용이 큽니다. 본 시스템은 이 과정을 Agent workflow로 연결하여 평가 준비 시간을 줄이고, 실험 이력과 평가 기준의 추적성을 강화합니다.

## 적용 예시

입력:

```text
시나리오 2 고속도로 cut-out 상황에서 V2X와 No V2X를 평가하고 KPI를 계산해줘.
```

출력:

- `scenario_definition.json`
- OpenCDA YAML/PY 복사본
- `run_manifest.json`
- KPI 계산 계획
- dashboard preview
- 평가 계획 보고서

## 최종 목표

평가기관이 자율주행차 출시 전 다양한 위험 시나리오를 반복 검증하고, 차량의 종합 성능을 정량적으로 비교할 수 있는 AI 기반 평가 프로그램을 구축하는 것입니다.
